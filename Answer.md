- 1️⃣ What is the difference between var, let, and const?
--> var is global scope. it can be called before initialization.
--> let is block scope. it cannot be accessed before initialization. variable can be reassigned
--> const (same as let) but cannot be reassigned.

- 2️⃣ What is the spread operator (...)?
--> it takes values from array and expand the array values into indiviual element.
- 3️⃣ What is the difference between map(), filter(), and forEach()?
--> map is a array method which do function on each array element and return new array
--> filter() gives new array based on condition it takes
--> forEach() works on every element of array but donot return new array

- 4️⃣ What is an arrow function?
--> its a way to write function using only parameter like this: const fun = (parameter) => {'function body'}
 - 5️⃣ What are template literals?
--> template literals `${}` is used to dynamically show innertext.
